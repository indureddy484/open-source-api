## 🚀 Getting Started

To run the project locally:

```bash
# 1. Clone the repository
git clone https://github.com/indureddy484/open-source-api.git

# 2. Navigate into the project directory
cd open-source-api

# 3. Install dependencies
npm install

# 4. Start the dev server
npm run dev
